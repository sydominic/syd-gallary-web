그림 배치 방법

지원 형식: PNG, JPG, JPEG

그림은 원본 가로세로 비율을 유지하여 액자 안에 맞춰집니다.
원본 비율과 액자 비율이 다르면 남는 공간에는 검은 받침면이 표시됩니다.
그림을 강제로 늘리거나 잘라내지 않습니다.

1. 지정 배치
아래 폴더에 그림을 넣으면 층과 전시유형을 사용자가 지정할 수 있습니다.
각 폴더 안에서는 파일명 오름차순으로 빈 슬롯에 배치됩니다.

1F/Main   : 2개
1F/Wall   : 2개
1F/Stair  : 4개
1F/Easel  : 6개

2F/Main   : 2개
2F/Wall   : 1개
2F/Stair  : 4개
2F/Easel  : 5개

3F/Main   : 2개
3F/Wall   : 1개
3F/Easel  : 4개

3층에는 별도의 Stair 슬롯이 없습니다.

전체 액자 수는 33개입니다.
메인 아트월의 왼쪽과 오른쪽 액자 모두 그림 업로드 대상입니다.

2. 자동 배치
Artworks 폴더 바로 아래에 넣은 그림은 지정 배치 후 남은 슬롯에 자동 배치됩니다.

3. 다시 불러오기
Play 실행 후 F5를 누르거나 화면의 "그림 + CSV 다시 불러오기" 버튼을 누르십시오.

4. 작품정보 입력
Artworks 폴더의 artworks.csv를 Excel에서 수정하십시오.

image_file : Artworks 폴더 기준 이미지 상대경로
title      : 작품명
artist     : 작가명
year       : 제작연도
description: 작품 설명

예:
1F/Main/mona.jpg,Mona Lisa,Leonardo da Vinci,1503,A portrait painting.

같은 파일명이 여러 폴더에 있을 수 있으므로 image_file에는 상대경로 입력을 권장합니다.
Excel에서는 "CSV UTF-8(쉼표로 분리)" 형식으로 저장하십시오.
쉼표, 큰따옴표, 여러 줄 설명이 포함된 셀도 처리됩니다.

선택 작품의 상세정보는 Play 중 I 키 또는 "작품 상세" 버튼으로 확인합니다.
CSV가 없거나 해당 행이 없으면 파일명을 작품명으로 대신 사용합니다.

작품 탐색:
L 키를 누르면 작품 목록, 검색, 층별/유형별 필터가 표시됩니다.
목록에서 작품을 누르면 해당 액자 앞으로 이동하고 상세정보가 열립니다.
M 키를 누르면 층별 미니맵이 표시됩니다.
미니맵의 작품 표시를 눌러서도 해당 액자 앞으로 이동할 수 있습니다.
미니맵이 열린 상태에서도 WASD, Shift와 마우스 시점 이동을 계속 사용할 수 있습니다.
Esc로 커서를 해제하면 이동을 유지하면서 미니맵의 층과 작품 표시를 클릭할 수 있습니다.

관람 동선:
G 키를 누르면 관람을 시작하거나 중지합니다.
N 키를 누르면 아직 관람하지 않은 다음 작품 앞으로 이동합니다.
직접 작품 가까이 걸어가거나 목록/미니맵으로 이동해도 관람 완료로 기록됩니다.
화면에서 관람 완료 수와 남은 작품 수를 확인할 수 있습니다.

데이터 진단 보고서:
Artworks/_reports/artwork_metadata_report.txt

실행파일명_Data/StreamingAssets/Artworks
